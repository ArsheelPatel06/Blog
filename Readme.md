## 🛠️ Quick Start

```bash
# Clone the repository
git clone https://github.com/ArsheelPatel06/Blog.git

# Navigate to project directory
cd Blog

# Open in VS Code
code .

# Or open index.html in browser
open index.html  # For macOS
```

## 📂 Project Structure

```
Blog/
```
